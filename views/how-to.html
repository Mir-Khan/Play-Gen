How to create a new playlist

  • Enter the name of the new playlist

  • Enter the number of songs you want in the playlist (max of 1000)

  • Decide if you want the playlist publicly visible

  • Press the submit button

How to modify an existing playlist

  • Enter the name of the playlist you want to modify EXACTLY as it is on Spotify

  • Enter the number of songs you want in the playlist (max of 1000)

  • Press the submit button

Some important notes

  • When using the modify tool, make sure you have at least one song in the playlist
    so the tool knows what to add

  • Since the creation tool depends on your listening history, it won't work well
    for a new Spotify account
